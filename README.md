## Getting Started

### Prerequisites

- PHP 7.4+
- Web server (Apache/Nginx)
- Modern web browser.
- Firebase account.

### Installation

1. Clone the repository:

   ```sh
   git clone https://github.com/VinayShetyeOfficial/hyper-chat-app.git
   ```

2. Start your local server:

   ```sh
   php -S 0.0.0.0:8000
   ```

3. Access the application:
   ```
   http://localhost:8000
   ```
